# testing-apk

This repository currently contains a decompiled Android Unity APK tree. Work has started toward `.exe` deliverables by adding both a no-device Windows desktop shell in `tools/windows_desktop_stub` and an emulator-backed Windows launcher in `tools/windows_launcher`.

See `tools/windows_launcher/README.md` for build and run instructions.
For step-by-step EXE build options, see `HOW_TO_BUILD_EXE.md`.
