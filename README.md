# testing-apk

This repository currently contains a decompiled Android Unity APK tree. Work has started toward an `.exe` deliverable by adding an emulator-backed Windows launcher in `tools/windows_launcher` and a PyInstaller spec in `packaging/windows`.

See `tools/windows_launcher/README.md` for build and run instructions.
For step-by-step EXE build options, see `HOW_TO_BUILD_EXE.md`.
